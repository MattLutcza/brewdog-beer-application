Matt's BrewDog Website Application

Welcome to Matt's BrewDog Website! Here you have the option of going through the entire beer list of BrewDog's renowned brews, and discovering how they're made. If you know what you're looking for, then you can also narrow your search to a particular subset of beers. Even if you don't know what you're interested in, you can leave it to us and we can pick a random beer to show you information about!
Full Beer List Search Beer Random beer Technical Stuff
This is just a simple Angular 8 application built by using the @angular/cli module and then doing 'ng new'. It is purely Angular 8 + CSS. There is no NodeJS or Express integration, because this application utilizes the PunkAPI to get BrewDog beer information. The PunkAPI is a simple REST API that provides a few different ways to retrieve BrewDog beer details.
